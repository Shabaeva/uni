Тема Университет Кастомные типы Кампусы (с подключением гугл карты), События (отображение в зависимости от даты), Профессоры и Программы. Установлены связи между кастомными постами, т.е при нажатии на программу, пользователь увидит в каком кампусе проходит программа и кто ее ведет. Мультипоиск сделан с помощью JS, для этого был кастомизирован WP API. Поиск по всем постам, в том числе кастомным. Все примененные фичи были использованы в реальных проектах.
